# 🧠 Mental Health Check-In Simulator

A console-based C++ application that helps users track moods, view mood history, receive motivational quotes, and perform relaxation exercises.

---

# 📌 Features

* ✅ Mood Check-In System
* ✅ Mood History Tracking
* ✅ Motivational Quotes
* ✅ Relaxation Exercises
* ✅ File Handling for Data Storage
* ✅ Console-Based User Interface

---

# 💡 Mood Options

* Happy
* Sad
* Angry
* Stressed
* Anxious
* Hopeful
* Excited
* Peaceful
* Tired
* Confused

---

# 🧘 Relaxation Exercises

## 1. 4-7-8 Breathing Exercise

Helps reduce stress and anxiety.

## 2. 5-4-3-2-1 Grounding Exercise

Improves focus and calms the mind.

## 3. Progressive Muscle Relaxation

Relaxes body muscles and reduces tension.

---

# 📂 OOP Concepts Used

## Encapsulation

Private data members are used for data security.

## Abstraction

Different classes handle different functionalities.

## Classes and Objects

The project is divided into multiple classes for modular programming.

---

# 📁 Classes Used

| Class Name                | Purpose                                  |
| ------------------------- | ---------------------------------------- |
| `User`                    | Stores user information and mood history |
| `MoodTracker`             | Takes mood input from user               |
| `CodeProvider`            | Provides motivational quotes             |
| `CalmingActivityProvider` | Provides relaxation exercises            |
| `Assistant`               | Controls the overall application         |

---

# 💾 File Handling

The project uses:

* `ifstream` → Reading files
* `ofstream` → Writing files

Mood history is stored in text files like:

```txt
mood_history_username.txt
```
